About ATP

ATP ( アデノシン三 リン酸 ) はアデニン ( 塩基 ) とリボース ( 糖 ) に３つのリン酸が結 合したヌクレオチドです。 A TPase という酵素で加水分解された時、１つのリン 酸が剥がれます。その時に放出されるエネルギー ( ATP1mol あたり 7 .3kcal ) を使 って、筋肉を動かしたり、細胞を分解したり、物質を合成したりとあらゆる生命 活動が行えます。私たちにとってなくてはならない物質なのです。 ここから 次に A TP の生成について説明 します 。 食物から得たエネルギーのうち、最も標準的なグルコースから ATP を生成する 方法で考えます。 食べ物から得たエ ネルギーを A TP に蓄えることを呼吸と言い、２段階の反応が あります。 ( ① 解糖系：グルコース→ピルビン酸：細胞基質内：酸素必要なし ) ( ② T CA 回路：ピルビン酸→二酸化炭素と水：ミトコンドリア o r 原核細胞膜： 酸素が必要 ) 生成 方法は さらに大きく ２つ ずつ あ り ます。 １つ目は、グルコースの反応中間体に A DP とリン酸が直接化合してできる「基 質レベルのリン酸化」 です。 ２つ目は、電子伝達系 を利用した 「酸化的リン酸化」 で す 。 呼吸で消費される物質から、酵素の力で水素と電子が取り出され、電子をミトコ ンドリアの内膜の「電子伝達系」まで運 びます ( 原核細胞の場合、 細胞 膜 の内側 ) 。 電子伝達系は電子を受け取ると、膜の内外での電位差を利用して A DP から A T P を 合成します。

2.1 細胞内シグナル伝達

細胞内では、外界の変化を入力として受け入れ、入力に応じた細胞変化が行われている。刺激を受け入れる器官を受容体と呼び、細胞膜の受容体に特定の刺激因子(リガンド)が結合すると、細胞内へ信号を送り細胞質にある様々な酵素を活性化させていく反応をおこす。この一連の流れを細胞内シグナル伝達という。最終的に標的因子が活性化されて入力に応じた細胞応答を起こすのである。つまり、細胞応答は細胞内シグナル伝達によって制御されているのである。

2.3 kholodenkoモデル

本研究では、細胞内シグナル伝達系のモデル化の際kholodenkoモデルを基にシミュレーションする。kholodenkoモデルはBorisN.KholodenkoがEGFのシグナル伝達をモデル化したものである。生化学実験のデータと比較して作られたモデルのため非常に信頼の高いモデルである。今回はATPを減少・消失させて、実際にRasたんぱく質をリン酸化させない事が可能かシミュレーションする。 作成した微分方程式を基にオイラー法によって、Rasタンパク質の時間あたりの濃度変化をグラフに表す。
